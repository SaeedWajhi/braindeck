Teach It Back
The Feynman technique — explain from memory, then compare
The most powerful learning technique
Richard Feynman said: if you can't explain it simply, you don't understand it yet. You'll see a concept name, write your explanation from memory — no notes — then compare with your saved answer. Gaps in your explanation reveal gaps in your understanding.

📋 Google Drive Setup Guide
One-time setup. Takes about 5 minutes.
1
Go to Google Cloud Console
Visit console.cloud.google.com and sign in.
2
Create a new project
Click "New Project", name it KnowledgeVault, click Create.
3
Enable Google Drive API
APIs & Services → Library → search Google Drive API → Enable.
4
Create OAuth credentials
Credentials → Create Credentials → OAuth client ID → Web application. Add your file's local path to Authorized JavaScript origins.
5
Copy Client ID
Copy the Client ID (looks like 123456.apps.googleusercontent.com) and paste it above.
